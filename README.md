# PC — Processing Core for TIMDER System

Repozytorium zawiera podstawowe klasy geometryczne i logiczne:
- Motion / Rotation / Twist — operatory ruchu i skrętu
- Tetroid / Triangle — struktury topologiczne
- Memory — zarządzanie stanem przejściowym
- SkretAI — warstwa przetwarzania informacji
- JCompressor — kompresja strukturalna danych

Cel: stworzenie przenośnego rdzenia dla systemu TIMDER.
